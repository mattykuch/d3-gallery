About This Portfolio

Welcome to my D3.js Visualization Portfolio! This website showcases a collection of interactive data visualizations created using D3.js, a powerful JavaScript library for producing dynamic, interactive data visualizations in web browsers.

My name is Matthew Kuch, and I am passionate about data visualization and storytelling through data. This portfolio serves as a platform to share my projects, insights, and experiences in the field of data visualization.

In this portfolio, you will find various visualizations, including bar chart races, line charts, and scatter plots, each demonstrating different aspects of data representation and interaction. I hope these projects inspire you to explore the world of data visualization!
